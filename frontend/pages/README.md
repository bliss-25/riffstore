# 🎸 RiffStore

RiffStore is a modern music instrument store and lesson booking platform.

Users can:
- Browse musical instruments
- Listen to instrument demos
- Purchase products
- Book music lessons with teachers

---

## 🚀 Features

- Dynamic landing page with guitar accordion hero section
- Instrument product catalog
- Music lesson booking system
- Shopping cart and checkout
- User accounts
- Teacher profiles

---

## 🛠 Tech Stack

Frontend:
- HTML
- CSS
- JavaScript
- GSAP animations

Backend:
- Java Spring Boot

- MySQL
